# Auto Socorro Liberato – Guincho em Ipatinga MG 24h

Este projeto é o site oficial do Auto Socorro Liberato, oferecendo serviços de guincho, reboque e transporte de veículos em Ipatinga e região.

## Tecnologias Utilizadas

Este projeto foi construído com:

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

## Como Executar o Projeto Localmente

Para trabalhar localmente, você precisará do Node.js e npm instalados.

Siga estes passos:

```sh
# 1. Clone o repositório
git clone https://github.com/7pessoal-source/liberato-auto-support.git

# 2. Navegue até o diretório do projeto
cd liberato-auto-support

# 3. Instale as dependências necessárias
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
```
